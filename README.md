# FUTURE_PE_01

## AI Website Copy Generator for Local Businesses

### Project Overview

This project was created as part of the Future Interns Prompt Engineering Internship.

The objective was to create an AI-generated website prototype and website content for a local business using structured prompts.

### Business Chosen

GD Cafe (Great Day Cafe)

### Location

Hyderabad, India

### Target Audience

* College Students
* University Students
* Freelancers
* Young Professionals

### Website Features

* Affordable Menu
* Study & Chill Zone
* Free Wi-Fi
* Charging Ports
* Student Discounts
* Online Ordering
* Responsive Design

### Tools Used

* ChatGPT
* Lovable AI
* GitHub

* AI-Assisted Design
* User Experience Thinking
